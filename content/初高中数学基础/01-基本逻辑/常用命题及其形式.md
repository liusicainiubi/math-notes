### 一、命题类型与否定规则

| 命题类型     | 逻辑符号               | 标准形式          | 🔴否定规则                                                                    |
| -------- | ------------------ | ------------- | ------------------------------------------------------------------------- |
| **全称命题** | $$\forall$$ (任意)   | 对所有 x，A 都成立   | 存在 x，使得 A 不成立 <br>例：原命题 "所有数都大于 0"<br>否定为 "存在一个数不大于 0"                    |
| **特称命题** | $$\exists$$ (存在)   | 存在 x，使得 A 成立  | 对所有 x，A 都不成立 <br>例：原命题 "存在一个数大于 0"<br>否定为 "所有数都不大于 0"                     |
| **蕴含命题** | $$A \Rightarrow $$ | 若 A 成立，则 B 成立 | A 成立 **且** B 不成立 <br>等价逆否命题：$$\bar{B} \Rightarrow \bar{A}$$（原命题与逆否命题同真同假） |

> ⚠️ 核心口诀：**否定的时候，全称变特称，特称变全称，只否定结论，条件不动**

#### 多元命题应用（极限定义）

考研高数核心定义，完全是这套逻辑的体现：

- 极限定义：$$\lim_{n \to \infty} x_n = a \iff \boldsymbol{\forall \varepsilon > 0},\boldsymbol{\exists} N > 0,\boldsymbol{\forall} n > N \implies |x_n - a| < \varepsilon$$    - 极限否定："a 不是数列的极限" $$\iff \boldsymbol{\exists \varepsilon > 0},\boldsymbol{\forall} N > 0,\boldsymbol{\exists} n > N \implies |x_n - a| \ge \varepsilon$$
### 二、核心符号全解（零基础必记）

> 所有符号本质都是正整数 / 正数，只是分工不同，再也不用混！
#### 1. ε：极限的精度标尺

- **本质**：希腊字母伊普西隆，代表**任意小的正数**，用来量化 "无限接近"
- **通俗类比**：你给数列提的 "精度要求"，要多准有多准，比如误差不能超过 0.01，ε 就取 0.01
- **核心属性**：
    - 一定是**大于 0 的正数**，不能等于 0
    - 是**任意给定**的，不是固定的某个数
- ⚠️ 考研易错点：真题错误选项常把$$\forall \varepsilon>$$（任意）改成$$\exists \varepsilon>$$（存在），一眼就能排除

#### 2. 小 n vs 大 N：极限里的两个序号

| 符号       | 身份      | 作用                          | 通俗类比                             |
| -------- | ------- | --------------------------- | -------------------------------- |
| **小写 n** | 排队的所有序号 | 数列里的第 n 个数，<br>是可以无限变大的变量   | 排队的所有人，1 号、2 号、3 号...            |
| **大写 N** | 固定的分界点  | 为了满足精度要求，<br>找出来的门槛，找出来就不变了 | 队伍里的分界线，从 N 号往后，<br>所有人都满足你的精度要求 |
##### 极限定义大白话翻译

> 你随便提一个多小的误差要求`ε`，我都能在数列里找到一个固定的分界点`N`。**只要序号****`n`****超过了这个分界点****`N`****（也就是 n>N），这些位置的数和 a 的误差，绝对比你提的****`ε`****还要小**。
> 
> 不管你把误差要求提得多苛刻，我都能找到这样的`N`，那这个数列的极限就是 a。

> [!faq]- 👉 点击展开：数列极限ε-N定义 数字例子 
> 我们以极限为0的数列 $\boldsymbol{x_n=\frac{1}{n}}$ 举例说明： 
> 1. 给定误差要求$\boldsymbol{\varepsilon=0.01}$，解不等式得$\boldsymbol{n>100}$，分界点$\boldsymbol{N}$取100，n>100后所有项与0的误差都小于0.01； 
> 2. 给定误差要求$\boldsymbol{\varepsilon=0.0001}$，解不等式得$\boldsymbol{n>10000}$，分界点$\boldsymbol{N}$取10000，n>10000后所有项与0的误差都小于0.0001。

### 三、题目的注意事项
1. **选择题陷阱**：
    
    1. 逆命题 / 否命题坑：原命题 "若 A 则 B" 是对的，选项的 "若 B 则 A"（逆命题）、"若非 A 则非 B"（否命题）全是错的，比如 "级数收敛→通项趋于 0" 是对的，反过来就是错的
        
    2. 全称命题坑：错误选项说 "所有情况都成立"，你只要举一个反例就能直接排除
        
2. **证明题思路**：

    1. 正面推不动，立刻转逆否命题 / 反证法，不要死磕
        
    2. 带 n 的问题，第一反应想数学归纳法，90% 都能用
### 四、例题
#### eg1：
命题 p：存在 a∈R 且 a$\neq$ 0，对任意 x∈R，均有 f(x+a)<f(x)+f(a) 恒成立。命题 q1​：f(x) 严格单调减少，且 f(x)>0 恒成立。命题 q2​：f(x) 严格单调增加，且存在 $x_0$​​ 使得 f($x_0$​)=0。需要判断 q1​、q2​ 是否为 p 的充分条件。

---

##### 分析 q1​ 是 p 的充分条件

1. **已知条件**：f(x) 严格单调减少，且 f(x)>0。
2. **取 a>0**：
    
    - 因为 f(x) 严格单调减少，所以当 x+a>x 时，f(x+a)<f(x)。
    - 又因为 f(a)>0，所以 f(x)<f(x)+f(a)。
    
3. **推导结论**：
    
    f(x+a)<f(x)<f(x)+f(a)
    
    即 f(x+a)<f(x)+f(a) 对任意 x∈R 成立。
4. **结论**：q1​ 能推出 p，故 q1​ 是 p 的充分条件。✅

---

##### 分析 q2​ 是 p 的充分条件

1. **已知条件**：f(x) 严格单调增加，且存在$x_0$​使得 f($x_0$​​)=0。
2. **取 a=$x_0$**：
    
    - 因为 f(x) 严格单调增加，且 f($x_0$​​)=0，所以$x_0$​​是函数从负到正的分界点，即 a=$x_0$​​<0。
    - 此时 x+a=x+$x_0$​​<x，由严格单调增加得 f(x+a)<f(x)。
    - 又 f(a)=f($x_0$​​)=0，所以 f(x)=f(x)+f(a)。
    
3. **推导结论**：
    
    f(x+a)<f(x)=f(x)+f(a)
    
    即 f(x+a)<f(x)+f(a) 对任意 x∈R 成立。
4. **结论**：q2​ 能推出 p，故 q2​ 是 p 的充分条件。✅
