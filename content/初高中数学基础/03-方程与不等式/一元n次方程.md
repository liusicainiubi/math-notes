> [!ABSTRACT] 1. 一元二次方程及其解
> 
> **标准形式**：$ax^2 + bx + c = 0 \quad (a \neq 0)$
> 
> **判别式**：$\Delta = b^2 - 4ac$
> 
> **根的情况判定**：
> 
> 1. **$\Delta > 0$**：有两个互异实根 $x_{1,2} = \frac{-b \pm \sqrt{\Delta}}{2a}$。
>     
> 2. **$\Delta = 0$**：有两个相等实根 $x_1 = x_2 = -\frac{b}{2a}$。
>     
> 3. **$\Delta < 0$**：在实数范围内无解（在复数范围内有一对共轭复根）。
>     

> [!TIP] 2. 重要方法与结论：韦达定理（常考！）
> 
> **(1) 一元二次方程根与系数的关系**：
> 
> 设 $ax^2 + bx + c = 0$ 的两根为 $x_1, x_2$，则：
> 
> - **两根之和**：$x_1 + x_2 = -\frac{b}{a}$
>     
> - **两根之积**：$x_1 \cdot x_2 = \frac{c}{a}$
>     
> - **两根之差的绝对值**：$|x_1 - x_2| = \frac{\sqrt{\Delta}}{|a|}$ （这个结论在处理根的间距时极快，建议背诵！）
>     
> 
> **(2) 一元 $n$ 次方程的推广**：
> 
> 设 $f(x) = a_n x^n + a_{n-1} x^{n-1} + \dots + a_0 = 0 \quad (a_n \neq 0)$：
> 
> - 所有根之和：$\sum x_i = -\frac{a_{n-1}}{a_n}$
>     
> - 所有根之积：$\prod x_i = (-1)^n \cdot \frac{a_0}{a_n}$
>     
> 
> > [!IMPORTANT] 考研笔记：
> > 
> > 一元 $n (n > 2)$ 次方程通常不直接求解，而是利用**换元法**降次后再解，或者利用韦达定理处理根的整体性质。

> [!EXAMPLE] 例 1：韦达定理与判别式的综合证明
> 
> **题目**：设函数 $f(x) = 3ax^2 + 2bx + c$，且满足 $a + b + c = 0$，$f(0)f(1) > 0$。
> 
> 证明：
> 
> (1) 方程 $f(x) = 0$ 有实根；
> 
> (2) $-2 < \frac{b}{a} < -1$；
> 
> (3) 若 $x_1, x_2$ 是方程 $f(x) = 0$ 的两个实根，则 $|x_1 - x_2| < \frac{2}{3}$。

> [!faq]- 👉 例 15：逻辑详解与证明（点击展开）
> 
> **(1) 证明有实根（即证 $\Delta > 0$）**
> 
> 首先证明 $a \neq 0$：若 $a=0$，则由 $a+b+c=0$ 得 $b+c=0$。此时 $f(0)f(1) = c(2b+c) = c(-2c+c) = -c^2 \le 0$，与已知 $f(0)f(1) > 0$ 矛盾，故 $a \neq 0$。
> 
> 计算判别式：$\Delta = (2b)^2 - 4(3a)(c) = 4b^2 - 12ac$。
> 
> 将 $c = -a-b$ 代入：$\Delta = 4b^2 - 12a(-a-b) = 4(b^2 + 3ab + 3a^2)$。
> 
> 配方：$\Delta = 4[(b + \frac{3}{2}a)^2 + \frac{3}{4}a^2]$。
> 
> 因为 $a \neq 0$，所以 $\Delta > 0$，方程必有两个互异实根。
> 
> **(2) 证明范围 $-2 < \frac{b}{a} < -1$**
> 
> 已知 $f(0)f(1) = c(3a+2b+c) > 0$。
> 
> 将 $c = -a-b$ 代入：$(-a-b)(3a+2b-a-b) = -(a+b)(2a+b) > 0$。
> 
> 即 $(a+b)(2a+b) < 0$。
> 
> 两边同时除以 $a^2$：$(1 + \frac{b}{a})(2 + \frac{b}{a}) < 0$。
> 
> 设 $t = \frac{b}{a}$，则 $(t+1)(t+2) < 0$，解得 $-2 < t < -1$。
> 
> **(3) 证明根之差 $|x_1 - x_2| < \frac{2}{3}$**
> 
> 由韦达定理得：$x_1 + x_2 = -\frac{2b}{3a}$，$x_1 \cdot x_2 = \frac{c}{3a}$。
> 
> 利用公式：$|x_1 - x_2| = \sqrt{(x_1 + x_2)^2 - 4x_1 x_2} = \sqrt{\frac{4b^2}{9a^2} - \frac{4c}{3a}}$。
> 
> 代入 $c = -a-b$ 并化简：
> 
> $|x_1 - x_2| = \sqrt{\frac{4}{9} (\frac{b^2}{a^2} + 3\frac{b}{a} + 3)} = \frac{2}{3} \sqrt{(\frac{b}{a} + \frac{3}{2})^2 + \frac{3}{4}}$。
> 
> 令 $t = \frac{b}{a}$，研究函数 $g(t) = (t + \frac{3}{2})^2 + \frac{3}{4}$ 在 $t \in (-2, -1)$ 上的取值。
> 
> 当 $t = -2$ 或 $t = -1$ 时，$g(t) = 1$；当 $t = -1.5$ 时，$g(t) = 0.75$。
> 
> 故在开区间 $(-2, -1)$ 内，$0.75 \le g(t) < 1$。
> 
> 从而 $|x_1 - x_2| < \frac{2}{3} \sqrt{1} = \frac{2}{3}$。证毕。 ✅
