> [!ABSTRACT] 一、 一元二次方程及其解的性质
> 
> **1. 标准形式**：$ax^2 + bx + c = 0 \quad (a \neq 0)$
> 
> **2. 判别式**：$\Delta = b^2 - 4ac$
> 
> - **$\Delta > 0$**：两个互异实根 $x_{1,2} = \frac{-b \pm \sqrt{\Delta}}{2a}$
>     
> - **$\Delta = 0$**：两个相等实根 $x_1 = x_2 = -\frac{b}{2a}$
>     
> - **$\Delta < 0$**：实数范围内无实根
>     

---

> [!TIP] 二、 重要结论与方法：韦达定理及其变形
> 
> **1. 韦达定理核心公式**：
> 
> 若 $ax^2 + bx + c = 0$ 的两根为 $x_1, x_2$：
> 
> - $x_1 + x_2 = -\frac{b}{a}$
>     
> - $x_1 \cdot x_2 = \frac{c}{a}$
>     
> - **两根之差绝对值（速算公式）**：$\mathbf{|x_1 - x_2| = \frac{\sqrt{\Delta}}{|a|}}$
>     
> 
> **2. 考研常考代数变形**：
> 
> - $x_1^2 + x_2^2 = (x_1+x_2)^2 - 2x_1x_2$
>     
> - $\frac{1}{x_1} + \frac{1}{x_2} = \frac{x_1+x_2}{x_1x_2} = -\frac{b}{c}$
>     
> - $(x_1-x_2)^2 = (x_1+x_2)^2 - 4x_1x_2$
>     

---

> [!EXAMPLE] 例 1：韦达定理与判别式的综合应用
> 
> **题目**：设 $f(x) = 3ax^2 + 2bx + c$，且满足 $a + b + c = 0$，$f(0)f(1) > 0$。
> 
> 证明：
> 
> (1) 方程 $f(x) = 0$ 有实根；
> 
> (2) $-2 < \frac{b}{a} < -1$；
> 
> (3) 若 $x_1, x_2$ 是方程 $f(x) = 0$ 的两个实根，则 $|x_1 - x_2| < \frac{2}{3}$。
> 
> > [!faq]- 👉 点击展开：解析过程
> > 
> > **(1) 证明有实根：**
> > 
> > - 计算 $\Delta = (2b)^2 - 4(3a)c = 4b^2 - 12ac$。
> >     
> > - 将 $c = -a-b$ 代入：$\Delta = 4(b^2 + 3ab + 3a^2) = 4[(b + \frac{3}{2}a)^2 + \frac{3}{4}a^2]$。
> >     
> > - 因为 $a \neq 0$，故 $\Delta > 0$，方程必有两个互异实根。
> >     
> > 
> > **(2) 证明范围 $-2 < \frac{b}{a} < -1$：**
> > 
> > - 已知 $f(0)f(1) = c(3a+2b+c) > 0$。
> >     
> > - 代入 $c = -a-b$：$(-a-b)(2a+b) > 0 \implies (a+b)(2a+b) < 0$。
> >     
> > - 两边除以 $a^2$：$(1 + \frac{b}{a})(2 + \frac{b}{a}) < 0$。
> >     
> > - 解不等式得：$-2 < \frac{b}{a} < -1$。
> >     
> > 
> > **(3) 证明 $|x_1 - x_2| < \frac{2}{3}$：**
> > 
> > - 利用速算公式：$|x_1 - x_2| = \frac{\sqrt{\Delta}}{|3a|} = \frac{2}{3} \sqrt{(\frac{b}{a})^2 + 3\frac{b}{a} + 3}$。
> >     
> > - 设 $t = \frac{b}{a} \in (-2, -1)$，研究函数 $g(t) = t^2 + 3t + 3$。
> >     
> > - 该函数在 $(-2, -1)$ 上的值域为 $[0.75, 1)$。
> >     
> > - 所以 $|x_1 - x_2| < \frac{2}{3} \sqrt{1} = \frac{2}{3}$。 ✅
> >
