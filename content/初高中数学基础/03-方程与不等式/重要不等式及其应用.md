### 一、 不等式的基本性质与绝对值不等式

> [!ABSTRACT] 1. 不等式的基本性质
> 
> ① 若 $a > b$，则 $a+c > b+c$
> 
> ② 若 $a > b, c > 0$，则 $ac > bc$
> 
> ③ 若 $a > b, c < 0$，则 $ac < bc$（**注意变号**）

> [!TIP] 2. 绝对值基本不等式（几何意义：距离）
> 
> 设 $a > 0$：
> 
> - **大于取两边**：$|x| > a \iff x > a$ 或 $x < -a$
>     
> - **小于取中间**：$|x| < a \iff -a < x < a$
>     
> 
> > [!faq]- 👉 点击展开：复杂绝对值不等式的转化
> > 
> > 1. **带上下限**：若 $b > a > 0$，则 $a \le |x| < b \iff a \le x < b$ 或 $-b < x \le -a$
> >     
> > 2. **整体代换**：把 $(ax+b)$ 看作一个整体，直接套用上面的“大于取两边，小于取中间”。
> >

### 二、 三大重要不等式（常考）

> [!IMPORTANT] 1. 均值不等式（调和 $\le$ 几何 $\le$ 算术 $\le$ 平方平均）
> 
> 对任意 $a, b > 0$，恒有：
> 
> $$\mathbf{\frac{2}{\frac{1}{a} + \frac{1}{b}} \le \sqrt{ab} \le \frac{a+b}{2} \le \sqrt{\frac{a^2+b^2}{2}}}$$
> 
> _(注：等号成立的条件均为 $a = b$)_

> [!TIP] 2. 三角不等式
> 
> 核心公式：$\mathbf{||a| - |b|| \le |a \pm b| \le |a| + |b|}$
> 
> > [!faq]- 👉 点击展开：取等号的条件（极易考选择题！）
> > 
> > - $||a| - |b|| = |a+b| \iff a \cdot b \le 0$
> >     
> > - $|a+b| = |a| + |b| \iff a \cdot b \ge 0$
> >     
> > - $||a| - |b|| = |a-b| \iff a \cdot b \ge 0$
> >     
> > - $|a-b| = |a| + |b| \iff a \cdot b \le 0$
> >     

> [!TIP] 3. 柯西不等式（常用于配凑求最值）
> 
> 一般形式：$(a_1^2 + \dots + a_n^2)(b_1^2 + \dots + b_n^2) \ge (a_1b_1 + \dots + a_nb_n)^2$
> 
> **考研最常用（二维形式）**：
> 
> $$\mathbf{(a_1^2 + a_2^2)(b_1^2 + b_2^2) \ge (a_1b_1 + a_2b_2)^2}$$

### 五、 区间恒成立问题与闭区间最值

> [!WARNING] 1. 二次不等式在区间 $(k_1, k_2)$ 上恒成立的条件
> 
> 设 $f(x) = ax^2 + bx + c > 0 \ (a \neq 0)$ 在区间 $(k_1, k_2)$ 上恒成立：
> 
> **① 当 $a < 0$ 时（开口向下）**
> 
> 只要保证区间的两个端点都在 $x$ 轴上方（或轴上）即可：
> 
> $$f(k_1) \ge 0 \quad \text{且} \quad f(k_2) \ge 0$$
> 
> **② 当 $a > 0$ 时（开口向上）**
> 
> 满足以下三种情况**之一**即可：
> 
> - **整体悬浮**：$\Delta < 0$
>     
> - **谷底在区间左侧**：$-\frac{b}{2a} \le k_1$ 且 $f(k_1) \ge 0$
>     
> - **谷底在区间右侧**：$-\frac{b}{2a} \ge k_2$ 且 $f(k_2) \ge 0$
>     

> [!TIP] 2. 二次函数在闭区间 $[m, n]$ 上的最值 $(a>0)$
> 
> 核心思路：**看对称轴 $x = -\frac{b}{2a}$ 是否在区间 $[m, n]$ 内。**
> 
> **① 对称轴在区间内** $\left(m \le -\frac{b}{2a} \le n\right)$
> 
> - **最小值**：在顶点处取得，$y_{\min} = f\left(-\frac{b}{2a}\right) = \frac{4ac-b^2}{4a}$
>     
> - **最大值**：在离对称轴较远的端点取得，$y_{\max} = \max\{f(m), f(n)\}$
>     
> 
> **② 对称轴在区间左侧** $\left(-\frac{b}{2a} < m\right)$
> 
> 此时函数在区间内**单调递增**：
> 
> - **最小值**：$y_{\min} = f(m)$
>     
> - **最大值**：$y_{\max} = f(n)$
>     
> 
> **③ 对称轴在区间右侧** $\left(-\frac{b}{2a} > n\right)$
> 
> 此时函数在区间内**单调递减**：
> 
> - **最小值**：$y_{\min} = f(n)$
>     
> - **最大值**：$y_{\max} = f(m)$
>
