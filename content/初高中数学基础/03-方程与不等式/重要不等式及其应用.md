### 一、 不等式的基本性质与绝对值不等式

> [!ABSTRACT] 1. 不等式的基本性质
> 
> ① 若 $a > b$，则 $a+c > b+c$
> 
> ② 若 $a > b, c > 0$，则 $ac > bc$
> 
> ③ 若 $a > b, c < 0$，则 $ac < bc$（**注意变号**）

> [!TIP] 2. 绝对值基本不等式（几何意义：距离）
> 
> 设 $a > 0$：
> 
> - **大于取两边**：$|x| > a \iff x > a$ 或 $x < -a$
>     
> - **小于取中间**：$|x| < a \iff -a < x < a$
>     
> 
> > [!faq]- 👉 点击展开：复杂绝对值不等式的转化
> > 
> > 1. **带上下限**：若 $b > a > 0$，则 $a \le |x| < b \iff a \le x < b$ 或 $-b < x \le -a$
> >     
> > 2. **整体代换**：把 $(ax+b)$ 看作一个整体，直接套用上面的“大于取两边，小于取中间”。
> >

### 二、 三大重要不等式（考研常考）

> [!IMPORTANT] 1. 均值不等式（调和 $\le$ 几何 $\le$ 算术 $\le$ 平方平均）
> 
> 对任意 $a, b > 0$，恒有：
> 
> $$\mathbf{\frac{2}{\frac{1}{a} + \frac{1}{b}} \le \sqrt{ab} \le \frac{a+b}{2} \le \sqrt{\frac{a^2+b^2}{2}}}$$
> 
> _(注：等号成立的条件均为 $a = b$)_

> [!TIP] 2. 三角不等式
> 
> 核心公式：$\mathbf{||a| - |b|| \le |a \pm b| \le |a| + |b|}$
> 
> > [!faq]- 👉 点击展开：取等号的条件（极易考选择题！）
> > 
> > - $||a| - |b|| = |a+b| \iff a \cdot b \le 0$
> >     
> > - $|a+b| = |a| + |b| \iff a \cdot b \ge 0$
> >     
> > - $||a| - |b|| = |a-b| \iff a \cdot b \ge 0$
> >     
> > - $|a-b| = |a| + |b| \iff a \cdot b \le 0$
> >     

> [!TIP] 3. 柯西不等式（常用于配凑求最值）
> 
> 一般形式：$(a_1^2 + \dots + a_n^2)(b_1^2 + \dots + b_n^2) \ge (a_1b_1 + \dots + a_nb_n)^2$
> 
> **考研最常用（二维形式）**：
> 
> $$\mathbf{(a_1^2 + a_2^2)(b_1^2 + b_2^2) \ge (a_1b_1 + a_2b_2)^2}$$
